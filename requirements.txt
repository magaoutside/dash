## Установка зависимостей

```bash
# Установить все зависимости
npm install

# Установить новую production зависимость
npm install package-name

# Установить новую development зависимость  
npm install -D package-name
```

## Используемые внешние ресурсы

- Google Fonts (Gilroy) - веб-шрифт
- Mocha CDN - статические ресурсы и изображения
- TON Connect - интеграция с TON блокчейном

## Примечания

- Все зависимости должны быть зафиксированы в package.json и package-lock.json
- Новые зависимости должны документироваться в этом файле с описанием их назначения
- При обновлении major версий зависимостей проверяйте совместимость
- Секреты MOCHA_USERS_SERVICE_* необходимы только для аутентификации через Google OAuth
